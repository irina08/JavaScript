Homework 6 Part 3
Current date and time.
Using String method split() for telephone numbers.

Click to Show/Hide Instructions
Create a webpage that does the following:

1. Using toString(), toLocaleString() and toUTCString() for the Date object, write the current date and time in these 3 formats. Using the getTimezoneOffset(), write the hours difference between PST or PDT (Pacific Standard Time or Pacific Daylight Time) and UTC, and the hours difference between EST or EDT (Eastern Standard Time or Eastern Daylight Time) and UTC. Note: EST is 3 hours ahead of PST (Pacific Standard Time). Note that for EDT (Eastern Daylight Time), the hours difference between EDT and UTC changes by one hour.

2. Input a telephone number as a string in the format (999) 999-9999. The script should use String method split to extract the area code as a token, the first 3 digits of the phone number as a token, and the last 4 digits of the phone number as a token. Display the area code in one text field and the seven-digit phone number in another text field. Make sure that when the user is typing the phone number it will automatically put the phone number in the (999) 999-9999 format - suggestion to use the jQuery mask for input.
